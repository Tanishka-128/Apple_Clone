 ## Apple Homepage Clone

This is a simple front-end clone of the Apple homepage built using HTML and CSS
for learning and practice purposes.


## Tech Stack
- HTML5
- CSS3

## Features
- Static Apple-style layout
- Responsive sections (basic)
- Uses relative paths for assets

## What I Learned
- Structuring HTML for real-world websites
- CSS layout and positioning
- Working with images and relative paths
- Using Git and GitHub for version control
